# Supported Versions

SanJuan AI is currently preparing for public beta.

## Support policy

| Version | Status | Supported |
| --- | --- | --- |
| `main` | Active development | Yes |
| `v1.0.0-beta` | Planned beta release | Yes, once released |
| Older commits | Historical | No |

## What support means during beta

During beta, support means:

- Documentation updates
- Bug fixes for local setup and CI
- Security fixes
- Retrieval safety improvements
- Source registry corrections
- Deployment guidance updates

It does not mean:

- Production uptime guarantees
- Official government support
- Guaranteed correctness for legal, tax, medical, emergency, immigration, court, police, or permit guidance

## Upgrade guidance

External testers should use the latest default branch until tagged beta releases begin.
